# notificationblservice的接口规范

**notificationblservice.Notification**

提供的接口

| 接口名称 | 语法   | 前置条件 | 后置条件 |
| ---- | ---- | ---- | ---- |
| notificationblservice.Notification.update | `public NotificationVO[] update();` | 用户已经登录。| 获得已有的通知。 |
| notificationblservice.Notification.acknowledge | `public ResultMessage acknowledge(NotificationVO notification);` | 用户已经登录，参数notification有效非null。| 已读已有的通知。|
| notificationblservice.Notification.activate | `public ResultMessage activate(NotificationVO notification);` | 用户已经登录，notification为已经通过审批的单据通知。| 激活单据。|
| notificationblservice.Notification.abandon | `public ResultMessage abandon(NotificationVO notification);` | 用户已经登录，notification为已经审批结束的单据通知。| 废弃单据。|
| notificationblservice.Notification.modify | `public ResultMessage modify(NotificationVO notification);` | 用户已经登录，notification为审批未通过的单据通知。|继续填写信息。| 记录准备修改的单据。|

需要的接口

| 接口名称 | 服务名 |
| ------- | ------| 
| `notificationdataservice.Notification.update()` | 更新通知。|
| `notificationdataservice.Notification.acknowledge(NotificationPO notification)` | 已读通知。|
| `logblservice.Logger.log(LogSeverity severity, String content);` | 记录日志 |

# draftblservice的接口规范

**draftblservice.DraftManagement**

提供的接口

| 接口名称 | 语法 | 前置条件 | 后置条件 |
| ------  | ---- | ------- | ------- | 
| draftblservice.DraftManagement.update | `public BillVO[] update();` | 用户已经登录。| 获得已经保存的草稿。|
| draftblservice.DraftManagement.delete | `public ResultMessage delete(BillVO bill);` | 用户已经登录，选择的单据处在草稿状态。| 删除选择的单据。|
| draftblservice.DraftManagement.resume | `public ResultMessage resume(BillVO bill);` | 用户已经登录，选择的单据处在草稿状态。| 记录准备继续填写的单据。|

需要的接口

| 接口名称 | 服务名 |
| ------- | ------| 
| `draftdataservice.DraftManagement.update()` | 更新草稿。|
| `draftdataservice.DraftManagement.delete()` | 删除草稿。|
